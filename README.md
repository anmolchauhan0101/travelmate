TravelMate is a full-stack MERN travel planning application that allows users to plan trips, explore destinations, find restaurants, hotels, and transport options.

## Features
- User authentication (JWT)
- Trip planning and management
- Explore destinations worldwide
- External data integration for restaurants & places
- Responsive UI with Tailwind CSS

## Tech Stack
**Frontend:** React, React Router, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**APIs:** OpenStreetMap, Google Maps (links)

## Setup Instructions

### Client
cd client  
npm install  
npm start  

### Server
cd server  
npm install  
npm run dev
